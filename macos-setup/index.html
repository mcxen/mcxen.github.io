概述

ML512G Mac mini 格式化重装后的恢复指南。备份文件位于 NAS //10.0.0.2/ML512G/20260625-MacBackup/。

一、系统基础设置

1.1 网络 & 代理

  • Clash Verge（clash-verge-rev）— 系统代理
  • 代理端口 127.0.0.1:7897（HTTP/SOCKS5）
  • Clash MAC helper 守护进程 /Library/LaunchDaemons/io.github.clash-verge-rev.clash-verge-rev.service.plist

1.2 系统偏好

  • 关闭”重新打开窗口”（NSQuitAlwaysKeepsWindows）
  • Dock 放左/下，自动隐藏
  • Finder 显示路径栏 + 扩展名
  • 三指拖拽（Accessibility → Pointer Control → Trackpad Options → Enable dragging → three finger drag）
  • 键盘重复速度最大
  • Caps Lock 映射为 Control 或切换输入法（看习惯）
  • 触控板轻点点击
