截图与录屏

Qx 内置截图、录屏、OCR、标注功能，无需安装第三方工具。

截图流程

1. 启动截图

方式 操作
快捷键 Alt+Shift+S（默认）
搜索 唤起 Qx → 输入「截图」→ Enter
托盘 菜单栏图标 → 截图

启动后：

  • 屏幕出现半透明暗色遮罩
  • 光标变为十字准星
  • Qx 自身界面隐藏（避免截到自己）

2. 选择区域

标准方式： 按住鼠标左键拖拽矩形。

加速方式（v0.6.11+）： 按住 Ctrl/⌘ 开始拖拽，松手直接截图并复制到剪贴板，跳过工具栏。

窗口吸附： 切换到窗口模式，悬停在窗口上方自动高亮，单击选中。

松手后选区出现调整手柄，可以拖动边缘微调大小和位置。

3. 确认截图

选区确定后，底部出现工具栏：

按钮 功能
截图 确认截图（自动复制到剪贴板）
录屏 开始录制选区
OCR → 剪贴板 识别文字，复制到剪贴板
OCR → 文本工具 识别文字，发送到内置文本编辑器
标注工具 箭头、文字、马赛克、序号等
取消 按 Esc 退出

快捷确认：

  • Enter — 截图进剪贴板
  • ⌘C — 截图进剪贴板
  • 双击选区 — 截图进剪贴板

4. 截图完成

截图后右下角弹出 Toast 通知，可点击：

  • 打开图片
  • 在 Finder/资源管理器中显示
  • 复制到剪贴板

截图自动保存到设置中指定的目录（默认 ~/Pictures/Qx/）。

5. 取消

方式 说明
Esc 随时取消，无副作用
工具栏关闭按钮 点 X 取消

重截上次区域（v0.6.11+）

截过一次后，可以快速重截同一区域，无需重新拖拽：

方式 操作
全局快捷键 Alt+Shift+R（默认关闭，需在设置中启用）
搜索 唤起 Qx → 输入「重截上次区域」→ Enter
Picker 内按 R 打开截图后按 R 键恢复上次选区

注意： 需要至少成功截过一次图，才会记录上次选区。

录屏

启动录屏

  1. 启动截图流程（快捷键或搜索）
  2. 拖拽选择录制区域
  3. 点击工具栏的录屏按钮

录制完成后自动生成 MP4/MOV 文件。

GIF 导出

录制完成后可在预览界面导出为 GIF 格式。

快捷键速查

快捷键 功能
Alt+Shift+S 开始截图
Alt+Shift+R 重截上次区域（需启用）
Alt+G 开始录屏
Esc 取消当前操作
Enter 确认截图
R Picker 内恢复上次选区
V Picker 内切换到录屏意图
Ctrl/⌘+拖拽 即时截图+复制

设置项

在 设置 → 截图与录屏 中可配置：

  • 截图保存目录
  • 自动复制到剪贴板（默认开启）
  • 截图延迟（秒）
  • 确认模式（精修/松手即截）
  • 录屏格式（MP4/MOV/GIF）
  • 全局快捷键自定义
