快速开始

快速开始

唤起与隐藏

操作 macOS Windows
唤起/隐藏 Qx ⌥ Space Alt + Space

按下快捷键，Qx 从后台弹出。再按一次隐藏。

基本导航

操作 macOS Windows
上下选择 ↑ ↓ ↑ ↓
确认/打开 Enter Enter
操作菜单 ⌘K Ctrl+K
返回/退出 Esc Esc

搜索

在搜索框中输入关键词，Qx 会实时搜索：

  • 应用 — 已安装的 App
  • 文件 — 本地文件（需要完全磁盘权限）
  • 命令 — 系统命令、别名
  • 计算 — 直接输入数学表达式

主界面

  • 左侧列表 — 搜索结果、功能入口
  • 右侧详情 — 选中项的详细信息
  • 底部状态栏 — 当前模式、快捷键提示

常用功能入口

唤起 Qx 后，直接输入关键词即可跳转：

关键词 功能
截图 打开截图工具
剪贴板 查看剪贴板历史
rss 打开 RSS 阅读器
天气 查看天气
设置 打开设置面板
