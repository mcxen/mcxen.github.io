项目结构

路径 内容
src/ React 前端，内置模块（剪贴板、RSS、截图、设置等）
src/plugin/ 插件注册、运行时 iframe、RPC、宿主端口
src-tauri/ Rust 核心：市场、剪贴板、截图、CLI/文件/HTTP 端口
public/doc/ 用户和开发者文档
docs/ 内部架构文档
scripts/ 检查脚本、Raycast 转换器、打包工具

开发规范

详见 AGENTS.md：

  • SOLID 原则在端口边界的应用
  • 跨平台 Rust 策略（#[cfg(target_os)] 分离）
  • 文件不超过 1000 行，按领域边界拆分
  • 公共接口变更必须同步文档
