macOS（推荐 Homebrew）

升级：

手动安装

从 GitHub Releases 下载：

平台 包格式 说明
macOS Apple Silicon .app.zip 解压拖入 /Applications，首次右键 → 打开
macOS Intel .app.zip 同上
Windows x64 NSIS 安装器 需要 WebView2

安装后 Qx 常驻菜单栏（macOS）或系统托盘（Windows）。

首次启动

macOS 首次启动会引导你开启以下权限：

  1. 完全磁盘访问 — 文件搜索功能必需
  2. 辅助功能（可选）— 剪贴板自动粘贴
  3. 屏幕录制（可选）— 截图与录屏功能
  4. 输入监控（可选）— 全局快捷键

每个步骤可跳过，后续在系统设置中手动开启。
