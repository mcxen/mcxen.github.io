Plugin System

Qx supports extending functionality through plugins. Plugins run in a sandbox environment and access host capabilities via context.* interfaces.

Install Plugins

From Marketplace

  1. Summon Qx → type “settings” → Enter
  2. Go to Extensions page
  3. Browse the plugin marketplace, click Install

Manual Install

Drag a .qx-plugin or .zip file into the Qx window, or click Import in settings.
