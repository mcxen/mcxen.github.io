Plugin System

Plugin System

Qx supports extending functionality through plugins. Plugins run in a sandbox environment.

Install Plugins

From Marketplace

Summon Qx → type “settings” → Enter → Extensions → Browse marketplace.

Manual Install

Drag a .qx-plugin or .zip file into Qx, or click Import in settings.

Core API

Interface Purpose Permission
context.cli Execute local commands cli
context.http HTTP requests http
context.storage Persistent storage storage
context.showNotification(msg) Show notification —
context.closePlugin() Close plugin —
