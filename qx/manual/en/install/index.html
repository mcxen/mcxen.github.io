After installation, Qx stays in the menu bar (macOS) or system tray (Windows).

First Launch

On macOS, the first launch guides you through permissions:

  1. Full Disk Access — Required for file search
  2. Accessibility (optional) — Auto-paste from clipboard
  3. Screen Recording (optional) — Screenshot & recording
  4. Input Monitoring (optional) — Global hotkeys

Each step can be skipped and enabled later in System Settings.
