Screenshot & Recording

Qx has built-in screenshot, screen recording, OCR, and annotation tools.

Screenshot Flow

1. Start Screenshot

Method Action
Hotkey Alt+Shift+S (default)
Search Summon Qx → type “screenshot” → Enter
Tray Menu bar icon → Screenshot

After starting:

  • A semi-transparent dark overlay appears
  • Cursor changes to a crosshair
  • Qx’s own UI hides (to avoid capturing itself)

2. Select Area

Standard: Hold left mouse button and drag a rectangle.

Fast mode (v0.6.11+): Hold Ctrl/⌘ while dragging — release instantly captures and copies to clipboard, skipping the toolbar.

Window mode: Switch to window mode, hover over a window to highlight it, click to select.

After releasing, resize handles appear — drag edges to adjust size and position.

3. Confirm Screenshot

After selecting, a toolbar appears at the bottom:

Button Function
Screenshot Confirm capture (auto-copies to clipboard)
Record Start recording the selected area
OCR → Clipboard Recognize text, copy to clipboard
OCR → Text Tool Recognize text, send to built-in text editor
Annotation tools Arrow, text, mosaic, numbered markers
Cancel Press Esc to exit

Quick confirm:

  • Enter — Screenshot to clipboard
  • ⌘C — Screenshot to clipboard
  • Double-click selection — Screenshot to clipboard

4. After Screenshot

A Toast notification appears in the bottom-right corner. You can click to:

  • Open the image
  • Show in Finder/Explorer
  • Copy to clipboard

Screenshots are saved to the configured directory (default ~/Pictures/Qx/).

5. Cancel

Method Notes
Esc Cancel anytime, no side effects
Toolbar close button Click X to cancel

Recapture Last Region (v0.6.11+)

After taking a screenshot, quickly recapture the same area without re-selecting:

Method Action
Global hotkey Alt+Shift+R (disabled by default, enable in Settings)
Search Summon Qx → type “recapture last region” → Enter
Press R in picker Open screenshot tool → press R to restore last selection

Note: You must have taken at least one screenshot before this works.

Screen Recording

Start Recording

  1. Start the screenshot flow (hotkey or search)
  2. Drag to select the recording area
  3. Click the Record button in the toolbar

After recording, an MP4/MOV file is generated automatically.

GIF Export

After recording, you can export as GIF format from the preview interface.

Quick Reference

Hotkey Function
Alt+Shift+S Start screenshot
Alt+Shift+R Recapture last region (must enable)
Alt+G Start recording
Esc Cancel current operation
Enter Confirm screenshot
R Restore last selection in picker
V Switch to recording intent in picker
Ctrl/⌘+drag Instant screenshot + copy

Settings

In Settings → Screenshot & Recording, you can configure:

  • Screenshot save directory
  • Auto-copy to clipboard (default: on)
  • Screenshot delay (seconds)
  • Confirm mode (refine / release)
  • Recording format (MP4/MOV/GIF)
  • Custom global hotkeys
