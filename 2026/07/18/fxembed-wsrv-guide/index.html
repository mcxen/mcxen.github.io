FxEmbed 与 wsrv.nl：两个低调但实用的网络工具

缘起

最近整理了一些日常用到的网络工具，其中 FxEmbed 和 wsrv.nl 是两个虽然知名度不高、但实际场景中非常趁手的服务。写篇文章记录一下用法和案例。

FxEmbed：让 X/Twitter 和 Bluesky 更好嵌入

FxEmbed 是一个开源项目，提供 fxtwitter.com、fixupx.com、fxbsky.app 等域名，用于优化 X/Twitter 和 Bluesky 帖子的嵌入展示。主要功能分为两类。

1. 帖子嵌入

当你把 X 链接粘贴到 Discord、Telegram 等平台时，FxEmbed 会生成更美观的嵌入卡片，支持主题色、媒体预览、翻译等。用法很简单——把域名从 x.com 换成 fxtwitter.com 或 fixupx.com 即可。

2. RSS / Atom 订阅

这是我最喜欢的功能。FxEmbed 可以为 X 和 Bluesky 用户生成 RSS 2.0 和 Atom 订阅源，无需 API token，直接在阅读器里关注用户动态。

订阅格式：

服务 类型 URL
X/Twitter 全部帖子 RSS https://fxtwitter.com/{handle}/feed.xml
X/Twitter 全部帖子 Atom https://fxtwitter.com/{handle}/feed.atom.xml
X/Twitter 仅媒体 RSS https://fxtwitter.com/{handle}/media.xml
Bluesky 全部帖子 RSS https://fxbsky.app/profile/{handle}/feed.xml
Bluesky 仅媒体 RSS https://fxbsky.app/profile/{handle}/media.xml

可选参数：?count=N（条目数，1-100）、?safe=1（隐藏敏感内容）、?lang=zh（翻译）。

Bluesky 的媒体源也很有用，只包含带图片/视频的帖子，适合做画廊风格的阅读体验。

3. URL 修饰符

FxEmbed 还提供一系列 URL 修饰符来控制嵌入行为：

  • d.{host} — 直接显示媒体文件
  • t.{host} — 纯文本模式，不加载媒体
  • g.{host} — 画廊模式
  • i.{host} — 即时查看模式
  • m.{host} — 马赛克敏感内容

wsrv.nl：图片缓存与缩放代理

wsrv.nl 是一个图片缓存/缩放服务。你传入图片 URL 和处理参数，它会自动抓取、处理，并通过 Cloudflare CDN 缓存分发。

核心用法

参数 说明
url 原始图片 URL（必需）
w 目标宽度（px）
h 目标高度（px）
fit 缩放模式：cover、contain、inside、outside
output 输出格式：jpg、png、webp 等

常见用途

  1. HTTPS 混合内容修复：给 HTTPS 网站嵌入 HTTP 图片
  2. 动态缩略图：不需要自己搭图片处理服务
  3. 格式转换：SVG/PDF 转 PNG 或 WebP
  4. CDN 加速：利用 Cloudflare 全球 300+ 节点缓存图片
  5. 动图支持：支持动态 WebP 和 GIF

支持格式：JPEG、PNG、BMP、GIF、TIFF、WebP、PDF、SVG。

总结

  • FxEmbed 解决了社会化媒体内容嵌入和订阅的问题，尤其是无需 API 的 RSS 订阅功能
  • wsrv.nl 是一个轻量的图片代理和处理方案，配合 CDN 效果很好

两个工具都是开源/免费服务，适合个人博客、自动化脚本、信息聚合等场景。
