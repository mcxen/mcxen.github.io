注意事项

  • 以上 API 全部免费、无需 Key、直接返回图片
  • 国内部分 API 可能因 CDN/备案问题偶尔不可用
  • Picsum 和次元API 最稳定，推荐优先使用
  • Lolicon 需要解析 JSON，适合做机器人或动态加载
  • 生产环境建议自建图床，这些作为备用
