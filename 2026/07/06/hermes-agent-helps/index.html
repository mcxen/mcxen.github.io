Hermes Agent 帮我做了些什么

最近深度体验了 Hermes Agent 这个 AI 助手，记录一下它帮我搞定的几件事。

🚀 项目发布一条龙

最频繁的工作就是给 Qx 项目做发布流程。从 v0.4.36 到 v0.4.57，中间发布了十多个版本：

  • 自动检测未提交的变更
  • 更新版本号（package.json、Cargo.toml、tauri.conf.json）
  • 运行 tsc --noEmit 和 cargo check 验证
  • 打 tag、推送到 GitHub
  • 一条命令搞定，再也不用手动敲 git add → git commit → git tag → git push

遇到网络波动推送失败会自动重试，省心很多。

🔧 frp 隧道配置

因为需要在外面连回家里的 Mac，它帮我：

  • 检查 frps 服务器是否在线
  • 配置 frpc 客户端
  • 开机自启 brew services start frpc
  • 全程不需要手动编辑配置文件

🤖 qodercli 自动化

Qx 项目里很多代码改动是通过 qodercli 完成的，Hermes 负责帮我操作：

  • 主题切换卡顿修复 — 排查 CSS 变量在暗黑模式下缺失的问题
  • Markdown 渲染升级 — 安装 react-markdown + shiki，后续换成更轻量的 @webc.site/math
  • 流式输出卡死修复 — 重构 Rust 后端的流式处理逻辑
  • 设置面板布局修复 — 导航项固定行高 + label 截断
  • 快捷键范围限制 — Cmd+, 只在搜索框焦点时才触发
  • NSPanel 悬浮面板 — macOS 原生无焦点面板实现

📦 资源下载

  • X/Twitter 视频 — 用 yt-dlp 下载指定推文视频
  • Facebook Astryx — 克隆并运行 Storybook 预览 Meta 的开源设计系统
  • 720yun 全景图 — 下载全景切片并合成为等距柱状投影图

🎯 小结

AI 助手已经从聊天玩具变成了真的能干活的生产力工具。从项目发布、代码重构、服务器配置到资源下载，基本上说”帮我搞一下”，它就能自己折腾出来。

这篇文章本身就是 Hermes 根据会话记录自动整理成文的。
