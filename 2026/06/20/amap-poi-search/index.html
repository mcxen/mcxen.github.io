给地图工具加上高德周边POI搜索

问题

几个月前我给 maps_client.py 接入了高德逆地理编码（reverse 命令），解决了中国地址只能到街道级的痛点。但 nearby 命令（周边POI搜索）一直用的 Overpass API（OpenStreetMap 数据），在中国小县城效果惨不忍睹。

之前在霞浦旅行时搜附近餐厅，霞浦松城 500m 范围内只返回了一家店，还因为 OSM 里 node + way 两条记录重复显示了三次。

方案：高德周边搜索 API

高德的 place/around 接口（周边POI查询）覆盖了中国全境的餐饮、住宿、医疗、购物等 46 个品类，数据质量远超 OpenStreetMap。

返回结构化 JSON，含店名、地址、电话、经纬度、距离、类型等字段。
