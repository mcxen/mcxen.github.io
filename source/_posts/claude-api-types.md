很多人不知道，Claude 的官方 API 其实分为三种来源，价格、可用范围和质量各不相同。

## 一、官方 API

**来源**：直接在 [console.anthropic.com](https://console.anthropic.com) 注册，绑定信用卡充值。

**价格**：按 token 计费，明码标价。比如 Claude Sonnet 4：
- 输入：$3 / 百万 token
- 输出：$15 / 百万 token

**特点**：
- 稳定、正规、有 SLA
- 需要实名认证（部分地区）
- 有速率限制，按充值金额提升
- 支持全部模型和功能

**适合**：企业、正规项目、需要稳定性的场景。

---

## 二、Claude Max 订阅反代

**来源**：订阅 Claude Max（$100/月 或 $200/月），通过第三方工具把网页版能力转成 API 接口。

**价格**：固定月费，不限量（有每日/每月上限）。

**特点**：
- 月费固定，用得多就划算
- 第三方反代工具，非官方支持
- 可能被封号（违反 ToS）
- 模型版本跟随网页版

**适合**：个人重度用户、想省钱的开发者。

---

## 三、逆向反代

**来源**：通过逆向工程模拟网页请求，或使用泄露的 session/token。

**价格**：通常极低甚至免费。

**特点**：
- 不稳定，随时可能失效
- 质量参差不齐
- 有安全风险（数据经过第三方）
- 可能使用被盗账号

**适合**：临时测试、不重要的场景。**不推荐用于生产。**

---

## 总结

| 类型 | 价格 | 稳定性 | 合规性 | 推荐度 |
|------|------|--------|--------|--------|
| 官方 API | 按量付费 | 高 | 完全合规 | ⭐⭐⭐⭐⭐ |
| Max 反代 | 固定月费 | 中 | 灰色地带 | ⭐⭐⭐ |
| 逆向反代 | 极低 | 低 | 违规 | ⭐ |

**建议**：正规项目用官方 API，个人折腾可以试试 Max 反代，逆向反代不推荐。
